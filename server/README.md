# 基于 Express.js 的后台管理系统

## 安装脚手架

```npm
npm install express-generator -g
```

## 生成项目

```npm
express myapp
```

## 进入项目

```npm
cd myapp
```

## 安装依赖

```npm
npm install
```

## 运行 app

```npm
SET DEBUG=myapp:\* & npm start
or
npm run start
```
